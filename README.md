## Expr1

### Условие

Как поменять местами значения двух переменных? Можно ли это сделать без временной переменной? Стоит ли так делать?

### Решение

**Способ 1 (рекомендуемый):**

Использование временной переменной:

```csharp
C = A;
A = B;
B = C;
```

Преимущества:

* высокая читаемость;
* отсутствие риска переполнения.

---

**Способ 2:**

Без временной переменной:

```csharp
A = A + B;
B = A - B;
A = A - B;
```

Недостатки:

* ухудшение читаемости;
* возможность переполнения числового типа.

## Expr2

### Условие

Задаётся натуральное трёхзначное число. Необходимо получить число, записанное теми же цифрами в обратном порядке.

---

### Способ 1. Преобразование строки

Число разбивается на отдельные символы:

```csharp
(A1, A2, A3) = (A3, A2, A1);
```

После чего символы преобразуются обратно в числа:

```csharp
A1 = int.Parse(A1.ToString());
A2 = int.Parse(A2.ToString());
A3 = int.Parse(A3.ToString());
```

И формируется итоговое значение:

```csharp
A = A1 * 100 + A2 * 10 + A3;
```

---

### Способ 2. Арифметический

Пример: `357`.

Последняя цифра:

```text
357 % 10 = 7
```

Первая цифра:

```text
357 / 100 = 3
```

Средняя цифра:

```text
357 / 10 = 35
35 % 10 = 5
```

После этого число собирается в обратном порядке.

## Expr3

### Условие

Задано время `H` часов (ровно). Найти угол между часовой и минутной стрелками.

Примеры:

```text
5 часов → 150°
20 часов → 120°
```

---

### Решение

На циферблате:

```text
360° / 12 = 30° на один час.
```

Переводим время в 12-часовой формат:

```csharp
H = H % 12;
```

Вычисляем два возможных угла:

```csharp
smallAngle = H * 30;
bigAngle = 360 - smallAngle;
```

Выбираем меньший:

```csharp
result = Math.Min(smallAngle, bigAngle);
```

## Expr4

### Условие

Найти количество чисел, меньших `N`, которые делятся на простые числа `X` или `Y`.

---

### Решение

Количество чисел, делящихся на `X`:

```text
(N − 1) / X
```

Количество чисел, делящихся на `Y`:

```text
(N − 1) / Y
```

Чтобы избежать двойного подсчёта:

```text
(N − 1) / (X × Y)
```

Итоговая формула:

```text
count = countX + countY − countXY
```

## Expr5

### Условие

Найти количество високосных лет на отрезке `[a, b]`, не используя циклы.

---

### Правила високосного года

Год является високосным, если:

* делится на 4;
* не делится на 100;

или

* делится на 400.

---

### Решение

Количество високосных лет до числа `N`:

```text
N / 4 − N / 100 + N / 400
```

Тогда для диапазона `[a, b]`:

```text
Leap(b) − Leap(a − 1)
```

## Expr6

### Условие

Посчитать расстояние от точки до прямой, заданной двумя различными точками.

---

### Решение

Используется формула площади треугольника:

```text
S = (1/2) × a × h
```

Отсюда:

```text
h = 2S / a
```

Площадь вычисляется через координаты вершин:

```text
S = (1/2) × |(X2 − X1)(Y1 − Y0) − (X1 − X0)(Y2 − Y1)|
```

Длина основания:

```text
a = √((X2 − X1)² + (Y2 − Y1)²)
```

## Expr7

### Условие

Найти вектор, параллельный и перпендикулярный прямой, заданной уравнением:

```text
Ax + By + C = 0
```

---

### Решение

Нормальный (перпендикулярный) вектор:

```text
(A, B)
```

Параллельный (направляющий) вектор:

```text
(−B, A)
```

или

```text
(B, −A)
```

## Expr8

### Условие

Дана прямая `L` и точка `A`.

Найти точку пересечения прямой `L` с перпендикуляром, проведённым из точки `A`.

---

### Решение

Пусть прямая задана точками:

```text
M1(X1, Y1)
M2(X2, Y2)
```

Точка:

```text
A(X0, Y0)
```

Направляющий вектор прямой:

```text
v = (X2 − X1, Y2 − Y1)
```

Вектор к точке:

```text
u = (X0 − X1, Y0 − Y1)
```

Коэффициент проекции:

```text
k = (u · v) / (v · v)
```

или

```text
k =
((X0 − X1)(X2 − X1) + (Y0 − Y1)(Y2 − Y1))
/
((X2 − X1)² + (Y2 − Y1)²)
```

Координаты искомой точки:

```text
Xp = X1 + k(X2 − X1)
Yp = Y1 + k(Y2 − Y1)
```
